AUDI B5 RS4 - VAG Option Codes


Every VAG car has a sticker in the Service Schedule book (as shown in the picture to the right) and somewhere inside the car (usually stuck in the boot) which has a large array of codes. Usually this is meaningless but with the help of the internet there is a superb list of what the codes actually mean. There are currently 12139 option codes in the list (updated 18th June 2011) and below are various formats of these option codes in PDF, Excel and Word.

There is also a tool that allows you to type in the 3 digit option codes (one at a time, or all together) and it will return a list of what they mean. Reall useful so give it a go!

VAG Option Code Tool
